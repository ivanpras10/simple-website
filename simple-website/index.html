HTML

HTML adalah singkatan dari "Hypertext Markup Language", yang dalam bahasa Indonesia dapat diartikan sebagai "Bahasa Penanda Hiperteks". HTML adalah bahasa markah atau markup language yang digunakan untuk membuat dan memformat konten halaman web. HTML digunakan untuk menentukan struktur dasar suatu halaman web, termasuk teks, gambar, tautan, dan elemen-elemen lainnya.

CSS

CSS merupakan singkatan dari "Cascading Style Sheets" atau "Lembar Gaya Cascading." Ini adalah bahasa lembar gaya yang digunakan untuk menggambarkan tampilan dan tata letak halaman web yang ditulis dalam HTML atau XML. CSS digunakan untuk menentukan bagaimana elemen-elemen sebuah halaman web harus ditampilkan, termasuk warna, font, jarak, posisi, dan lainnya.

Javascript

JavaScript adalah bahasa pemrograman tingkat tinggi yang paling sering digunakan untuk mengembangkan aplikasi web interaktif. Ini adalah bahasa skrip yang dapat dijalankan di sisi klien (di browser) atau di sisi server (misalnya dengan menggunakan Node.js). JavaScript memungkinkan pengembang untuk menambahkan berbagai jenis fungsionalitas interaktif ke situs web, seperti validasi formulir, animasi, manipulasi DOM (Model Objek Dokumen), dan interaksi dengan server.
